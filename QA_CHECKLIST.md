# QA Checklist

배포 전/기능 수정 후 최소 점검 항목입니다.

## 1) 서버 기동

- [ ] `backend` 실행: `npm start`
- [ ] `frontend` 실행: `npm start`
- [ ] 백엔드 헬스체크 확인: `http://localhost:5001/health` 응답 `ok: true`
- [ ] 프론트 접속 확인: `http://localhost:3000`

## 2) 업로드

- [ ] 폴더 업로드 버튼으로 이미지 폴더 업로드 가능
- [ ] 업로드 진행바가 표시되고 완료 후 사라짐
- [ ] 업로드 후 라이브러리 목록에 책이 보임

## 3) 페이지/미리보기

- [ ] 책 선택 시 페이지 목록 로드됨
- [ ] 썸네일 클릭 시 메인 미리보기 이미지 변경됨
- [ ] 전체 선택/해제 체크 동작 정상

## 4) OCR 공정

- [ ] 선택 면 공정 실행 가능
- [ ] 책 전체 공정 실행 가능
- [ ] 공정 진행바가 0% -> 100%로 점진 상승
- [ ] 완료 시 `공정 완료되었습니다` 버튼 표시

## 5) 다운로드

- [ ] 페이지별 `txt 다운로드` 시 `.txt` 파일로 저장됨
- [ ] `OCR 합본 다운로드` 시 `*_ocr_all.txt`로 저장됨
- [ ] 새 창/새 탭이 뜨지 않고 브라우저 기본 다운로드로 동작

## 6) 예외 상황

- [ ] 책 미선택 상태에서 공정 실행 시 안내 문구 표시
- [ ] 선택 면 0개 상태에서 선택 공정 실행 시 안내 문구 표시
- [ ] 네트워크/서버 오류 시 에러 배너 표시

## 7) 데이터 정합성

- [ ] OCR 완료 페이지가 처리 상태(색상/표시)로 구분됨
- [ ] 새로고침 후에도 처리 상태가 유지됨
- [ ] 책 삭제 시 목록/선택 상태가 정상 갱신됨

## 8) Git 점검

- [ ] `git status`가 의도한 변경만 포함
- [ ] `node_modules`, `build`, `uploads`, `temp` 등 생성물 미추적 확인
